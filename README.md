# Pixelol 🔷

> A browser-based pixel art editor with isometric diamond-shaped pixels. No installation. No dependencies. Open and draw.

## 🚀 Quick Start

**[▶ Open in Browser](https://lexbayart.github.io/pixelol/)**

Or download `index.html` and open it locally — it works completely offline.

## ✨ Features

- Isometric diamond-shaped pixel grid
- Full layer system: pixel layers, reference layers, folders
- Drag-and-drop layer reordering with folder support
- Stroke effects with color, width, style, and position control
- Sketch mode (Alt + pen) for non-destructive overlays
- Clipping layers, layer opacity, layer locking
- Undo / redo history with snapshot system
- Select tool with transform support for reference layers
- Crop tool for reference images
- PNG and SVG export
- Palette extraction from dropped images
- Audio feedback (SFX on interactions)
- Guides, grid, zoom, pan
- Invisibility mode for distraction-free drawing
- Fully offline — zero external dependencies at runtime

## 🛠️ How to Use

1. Open the app in any modern browser
2. Choose a tool from the left toolbar (pen, eraser, fill, eyedrop, line, rect, select)
3. Draw on the isometric canvas
4. Manage layers in the right panel — add, reorder, group into folders
5. Export your work as PNG or SVG

## 📦 Tech Stack

Vanilla JavaScript · HTML5 Canvas · CSS3 · Single HTML file · No frameworks · No build step
